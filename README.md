# 🕒 Digital Clock – DOM Practice Project

A lightweight, real-time digital clock built to practice fundamental JavaScript DOM manipulation, CSS positioning, and asynchronous timing events. 

This project captures live time data from the user's system and dynamically updates the user interface every second without reloading the page.

---

## 🚀 Features

* **Real-Time Updates:** Updates seamlessly every second using JavaScript timing events.
* **Smart Formatting:** Uses zero-padding (e.g., `05` instead of `5`) for a clean, consistent digital display.
* **12/24 Hour Toggle:** (Optional expansion) Easy to configure for AM/PM or military time formats.
* **Responsive Design:** Centered and styled using CSS Flexbox/Grid to look great on screens of all sizes.
